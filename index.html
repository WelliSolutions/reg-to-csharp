REG2C#

Converts a .REG file to C# code

Drag and drop your file onto the browse button.

How it works

This tool works completely in your browser just using Javascript. So make sure you haven't disabled it!

About

I adapted this tool, because I had a need for converting a .REG file to C#, because doing it manually is wasted time, which you could invest better.
