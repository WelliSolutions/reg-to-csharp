Reg-to-csharp

Provides an online converter from .reg files to C# code

View on GitHub Download .zip Download .tar.gz

Welcome to the GitHub Pages of the reg-to-csharp tool.

This tool makes it really easy for you to convert an convenient .reg file to ready to use C# code.

How it works

This tool works completely in your browser just using Javascript. So make sure you havn't disabled it!

About

I wrote this tool, because I once had a need for converting a .reg file to C#, because if you do it by hand it's useless wastet time, which you could invest better. If you like this tool, feel free to leave a star on my Github project page!

If you would like a new feature or just want to inform me about an bug, feel free to create an issue for that on Github!
